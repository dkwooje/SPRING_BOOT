Spring Boot를 처음 만들어 봅니다.
Spring Boot를 만들기 전에 IntelliJ 도구 사용법과 lombok 그리고 데이터베이스와 연동방법을 공부했습니다.
gradle.build 등 Spring Boot의 주요기능을 활용법을 배웠습니다.

Spring boot의 IntelliJ 설정을 배웠다면 본격적으로 코딩 공부를 합니다.
사용 코딩 기법은 타임림프기법입니다.
